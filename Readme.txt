Enhances WordPress admin UI with bulk actions, filters, and LearnPress/Eduma support. Built by George Tumanishvili.

== Description ==
This plugin enhances the WordPress admin UI with sortable columns, bulk actions, editor upgrades, and powerful filters for posts, pages, and LearnPress courses. Useful for course platforms, editorial teams, and content-heavy sites.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin via WordPress Admin > Plugins
3. That’s it! No configuration required.

== Frequently Asked Questions ==
= Does this plugin affect frontend users? =
No — all features are strictly admin-only.

= Is it compatible with the Eduma theme and LearnPress plugin? =
Yes, it is designed specifically with them in mind.
